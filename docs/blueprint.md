# **App Name**: NewsFlash

## Core Features:

- Authentication: Enable users to securely log in or sign up, storing authentication data in Firestore.
- User Profile: Allow users to manage their data, preferences, and log out. Persist data in Firestore.
- News Feed: Display a personalized news feed based on user preferences, fetched and displayed.
- Browse News: Enable users to search for news headlines and access full articles within the app using an in-app browser.
- AI News Chat: Implement an AI-powered chat feature (using the Gemini API) that allows users to discuss news articles. The AI tool can use information provided in articles.
- News Categories: Allow users to select news preferences by category, stored and managed in Firestore.
- YouTube News: Show the news videos from channels from YouTube, updated automatically.
- Reading history: Allows user to see past read articles and continues reading those article . Store into Firestore.
- Trending News: Provide a section that displays the current trending articles.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to convey authority and trust.
- Background color: Light grayish-blue (#E6E9ED), desaturated and subtle.
- Accent color: Soft violet (#8A2BE2) to highlight important interactions.
- Headline font: 'Space Grotesk' sans-serif, for a techy and modern feel; body font: 'Inter' sans-serif, for longer text. 
- Use simple, modern icons from a UI library, consistently styled.
- Employ a card-based layout for news articles to maintain a clean, organized presentation.
- Incorporate subtle transitions when loading news content for a smoother user experience.
